# Portfolio Generator - PRD

## 프로젝트 개요
개발자의 프로젝트를 입력하면 자동으로 아름다운 포트폴리오 웹사이트를 생성하는 도구.

## 핵심 기능

### 1. 프로젝트 관리
- 프로젝트 추가/편집/삭제 (이름, 설명, 기술스택, 링크, 이미지)
- 메타데이터 JSON 기반
- 기본 프로젝트 8개 사전 입력 (우리가 만든 시각화 프로젝트들)

### 2. 타임라인 뷰
- 프로젝트 진화 히스토리 시각화
- 버전 간 비교 (v1 → v2 → v3 ...)
- 날짜 기준 정렬
- 코드량 성장 차트

### 3. 코드 스니펫 하이라이팅
- JavaScript/HTML/CSS/Python 기본 문법 하이라이팅
- 복사 버튼
- 줄번호 표시
- 다크/라이트 테마

### 4. 라이브 데모
- iframe으로 데모 임베드
- 또는 스크린샷 갤러리
- 전체화면 보기 버튼

### 5. 반응형 + 다크 테마
- 모바일 최적화
- 다크/라이트 테마 토글
- 부드러운 스크롤 + 섹션 애니메이션

### 6. Export
- PDF 다운로드 (window.print 기반)
- PNG 스크린샷
- JSON 백업/복원

## 기술 스택
- 순수 HTML/CSS/JS (싱글 파일)
- 한국어 UI
- 다크 테마 기본
- 외부 라이브러리 금지
