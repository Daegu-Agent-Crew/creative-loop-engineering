화투패 구성

화투는 12개월 각 4장, 총 48장으로 구성됩니다.

  • 광(光) - 5장 (1월, 3월, 8월, 11월, 12월)
  • 띠(帶) - 10장 (홍단, 청단, 초단)
  • 열끗(동물) - 9장
  • 피 - 24장 (쌍피 포함)

게임 진행

  • 각 플레이어에게 10장씩 배분, 바닥에 8장
  • 자기 턴: 패 1장 내기 → 덱에서 1장 뒤집기
  • 같은 월의 패가 바닥에 있으면 가져감
  • 7점 이상이면 "고" 또는 "스톱" 선택

점수 계산

  • 광: 3광=3점, 4광=4점, 5광=15점, 비광=2점
  • 띠: 홍단/청단/초단 각 3점, 띠 5장=1점(+1장당 +1)
  • 열끗: 5장=1점 (+1장당 +1)
  • 피: 10장=1점 (+1장당 +1)
  • 고도리: 2,4,8월 열끗 = 5점

특수 규칙

  • 폭탄: 같은 월 3장 보유 시 한번에 내기
  • 뻑: 3번 연속 뻑 → 상대방 벌칙
  • 자뻑: 내가 낸 패와 뒤집은 패가 같은 월
  • 흔들기: 같은 월 3장 보유 시 선택 가능, 점수 2배
  • 총통: 같은 월 4장 보유 시 즉시 승리
  • 피박: 상대 피 7장 이하 → 점수 2배
  • 광박: 상대 광 0장 → 점수 2배
