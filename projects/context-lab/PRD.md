# OpenClaw Context Lab - "컨텍스트를 직접 만져보세요"

이전 시각화들은 에이전트의 컨텍스트를 **보여주는 것**이었다면, 이번엔 **직접 조작해보는 것**입니다. Bootstrap Files를 토글하고, 툴 스키마를 제거하고, 프롬프트를 편집하면서 토큰 변화를 실시간으로 관찰하는 인터랙티브 실험실.

---

## 📋 핵심 컨셉

**"읽기만 하지 말고, 만져보세요"**

Q: "TOOLS.md가 54KB나 되는데, 실제로 얼마나 중요할까?"
→ 토글 OFF 해보면 알 수 있습니다.

Q: "Compaction 전후로 얼마나 달라질까?"
→ Before/After 비교 버튼으로 확인하세요.

Q: "메모리가 꽉 차면 어떻게 되지?"
→ 직접 대화를 채워가며 확인해보세요.

## 🔧 인터랙티브 기능 (이전 버전과의 핵심 차이!)

### 1. 토글 ON/OFF (각 섹션별)

```
사용자가 좌측에서 체크박스를 토글:
- ✅ → 해당 섹션이 Live Preview에 포함
- ❌ → 제외됨 (회색, 줄긋기)
- 토큰 카운터 실시간 업데이트
- Context Window 게이지 변화

애니메이션: 토글 OFF 시 해당 블록이 축소되며 사라짐
```

### 2. 섹션 순서 드래그앤드롭

```
Bootstrap Files 순서 변경 가능:
SOUL → AGENTS → USER → IDENTITY → TOOLS → HEARTBEAT

순서를 바꾸면:
- Live Preview에 실시간 반영
- "섹션 순서가 LLM 동작에 미치는 영향" 팁 표시
```

### 3. 직접 편집

```
SOUL.md, AGENTS.md 등을 클릭하면:
- 인라인 에디터 열림
- 텍스트를 수정하면 토큰 카운터 즉시 업데이트
- "저장" → 변경사항 반영
- "되돌리기" → 원본 복원
```

### 4. 대화 추가 시뮬레이션

```
하단 입력창에 메시지를 타이핑:
→ Enter를 누르면 Session History에 추가
→ Assistant 자동 응답 시뮬레이션 (간단한 규칙 기반)
→ Tool Call이 필요한 질문이면 자동으로 Tool Call/Result 추가
→ Context Window가 점진적으로 채워짐

특별 메시지:
- "날씨" → web_search Tool Call 자동 추가
- "이미지 분석" → image Tool Call 자동 추가
- "기억해" → Memory Flush 트리거
```

## 🎨 시각 효과

### 토글 애니메이션
- ON→OFF: 해당 블록이 오른쪽으로 슬라이드 아웃 + 투명해짐
- OFF→ON: 왼쪽에서 슬라이드 인
- 토큰 카운터가 실시간으로 카운트업/다운

### Context Window 게이지 애니메이션
- 풀 차면 물이 차는 것처럼 위에서 아래로 채워짐
- 색상 그라데이션: 초록→노랑→주황→빨강
- 95% 도달 시 화면 가장자리 빨간 펄스

### Compaction Split View
- 왼쪽: Before (전체 대화)
- 오른쪽: After (요약 + 최근)
- 가운데 화살표 애니메이션

### 히트맵
- 큰 섹션은 밝은 색, 작은 섹션은 어두운 색
- 클릭하면 상세 정보
