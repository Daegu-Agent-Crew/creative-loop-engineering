# Agent Core Viz 4: 🫀 인체 해부학

OpenClaw Agent Core를 인체의 장기 시스템으로 시각화합니다.

기반 정보: CONTEXT.md 읽을 것.
반드시 $(pwd)/anatomy.html 파일에 저장할 것 (터미널 출력 금지).
외부 라이브러리 사용 금지 (순수 HTML/CSS/JS).

## 컨셉

Agent Core의 5개 서브시스템을 인체 장기에 매핑하여 시각화.

## 구현

### 장기 매핑
```
🧠 두뇌 = Agent Runtime (pi-mono) - 사고/추론
👁️ 눈/귀 = Channel Adapters - 입력 감각
🫁 폐 = Queue - 숨결(메시지 흐름)
❤️ 심장 = Session Manager - 맥박(세션 관리)
🛡️ 면역 = Tool Policy - 보호/필터링
💾 해마 = Memory System - 기억 저장
```

### 중앙 인체 시각화
CSS로 간단한 인체 실루엣. 장기 위치에 아이콘 배치.
활성화 시 장기가 밝아지고 맥박/호흡 애니메이션.

### 시뮬레이션
메시지 처리를 신체 반응으로:
1. 👁️ 감각 수신 (메시지 도착)
2. 🫁 호흡 (Queue 처리)
3. ❤️ 맥박 상승 (Session 로드)
4. 🧠 두뇌 활동 (추론 - 뇌파 그래프 애니메이션)
5. 🛡️ 면역 검사 (Tool Policy)
6. 🧠 다시 생각 (툴결과 처리)
7. 👁️ 반응 (응답 전송)

### 바이탈 사인 모니터 (오른쪽)
- 심박수 (메시지 처리 속도)
- 혈압 (Context Window 압력)
- 체온 (모델 부하)
- 호흡수 (Queue 처리량)
- EEG (툴호출 빈도)

### 상세 정보 (클릭)
장기 클릭 시 해당 서브시스템:
- 역할 설명
- Hook 포인트
- 관련 파일 (Bootstrap Files)
- 설정 옵션

### Memory 시각화
해마(기억) 클릭 시:
- 단기기억 (context window) ↔ 장기기억 (MEMORY.md) 다이어그램
- 벡터 검색 애니메이션 (의미 검색 시 신경망 빛남)
