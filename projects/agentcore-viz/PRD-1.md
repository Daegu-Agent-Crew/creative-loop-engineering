## 컨셉

뇌 단면도처럼 Agent Core의 5개 서브시스템을 신경망 뉴런으로 표현.

## 구현

### 중앙 뇌 구조
- 뇌 중앙에 Agent Runtime(pi-mono)이 위치
- 5개 뇌 영역으로 서브시스템 분배:
  - 전두엽(Channel Adapters) - 입력 처리
  - 두정엽(Session Manager) - 기억/인지
  - 측두엽(Queue) - 순서/리듬
  - 후두엽(Tool Policy) - 필터링
  - 뇌간(Agent Runtime) - 핵심 추론

### 시뮬레이션
"사용자가 '날씨 알려줘'라고 물었을 때" 시나리오:
1. 시냅스(연결선)이 전두엽에서 뇌간으로 빛나면서 신호 전달
2. 뇌간이 밝아지며 "추론 중..." 
3. 후두엽이 점등하며 툴호출 필터링
4. 뇌간에서 다시 시냅스가 번쩍이며 응답 생성
5. 전두엽으로 응답 전달

### 인터랙션
- 뇌 영역 호버 → 해당 서브시스템 설명
- 클릭 → 상세 정보 패널 (역할, 구성, Hook 포인트)
- 하단 컨트롤: 시나리오 선택, 속도, 리셋

### 비주얼
- 배경: 진한 파란색(#0a0a2e)에 뇌 윤곽 CSS 그라데이션
- 뉴런: 원형 노드, 활성화 시 빛나는 글로우
- 시냅스: 곡선 연결, 신호 전달 시 입자 이동
- 색상: 각 영역마다 다른 네온 색

### 정보 패널
오른쪽에 세로 패널:
- Agent Loop 단계별 설명
- Context Window 구성 (도넛 차트)
- 현재 활성 Hook 표시
