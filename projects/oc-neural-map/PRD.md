## 🎬 Cortex Theater (메인 스테이지) - 핵심!

### 영감: OpenClaw Office의 2D 오피스 + Speech Bubbles

### 시각화 1: Thought Cascade (생각 폭포)

```
사용자가 메시지를 보내면, 생각이 폭포처럼 흘러내림:

    💭 "날씨 알려줘"
         │
         ▼ (1.2초)
    ┌──────────┐
    │ 🧩 의도  │ ← 큐브가 회전하며 등장
    │   분석   │    "정보요청/위치필요"
    └────┬─────┘
         │ 시냅스 라인 (그라디언트 + 파티클)
         ▼ (0.8초)
    ┌──────────┐
    │ 📚 컨텍 │ ← 책이 펼쳐지는 애니메이션
    │   스트   │    SOUL.md, USER.md, memory/
    │   수집   │    각 파일이 카드로 펼쳐짐
    └────┬─────┘
         │
         ▼ (0.5초)
    ┌──────────┐
    │ 🔍 기억  │ ← 검색 바가 슬라이드
    │   검색   │    벡터 인덱스 스캔 애니메이션
    │   0.89   │    유사도 게이지
    └────┬─────┘
         │
         ▼ (2.3초)
    ┌──────────┐
    │ ⚡ 추론  │ ← 전기 스파크 효과
    │   모델   │    "과거 기록 있음→스킬 사용"
    │   GLM-5  │    TTFT 프로그레스
    └────┬─────┘
         │
         ▼ (0.3초)
    ┌──────────┐
    │ 🛠️ 도구  │ ← 툴 아이콘이 튀어나옴
    │   호출   │    wttr.in → 결과 카드
    │   wttr.in│    날씨 데이터 파싱 애니메이션
    └────┬─────┘
         │
         ▼ (1.5초)
    ┌──────────┐
    │ 💬 응답  │ ← 말풍선이 팽창
    │   생성   │    텍스트가 타자기처럼 나타남
    │   500tok │    (OpenClaw Office Speech Bubble 스타일)
    └──────────┘
```

**핵심:** 각 단계가 **입체 카드**로 등장하며, 단계 사이를 **시냅스 라인** (빛나는 파티클이 흐르는 곡선)이 연결

```
  ┌────────────────────────────────┐

  │                                │

```

**아바타 상태 애니메이션 (Office 참고):**
- `idle`: 살짝 흔들림, 눈 깜빡임
- `thinking`: 머리 위 💭 말풍선, 뇌파 파동
- `tool_calling`: 손이 도구를 만지는 모션
- `speaking`: 말풍선이 팽창/수축
- `error`: 빨간 글로우, 흔들림

**아바타는 에이전트 ID에서 결정론적으로 생성 (Office 참고)**

### 시각화 3: Collab Lines (영감: OpenClaw Office 협업 라인)

```
  서브에이전트 생성 시:

  👼 천사2 ═══════ 🤖 Claude ═══════ 🤖 Gemini
  (main)    ──────→ (codex)    ──────→ (review)
              "코드 작성해"    "리뷰해줘"
               150ms           230ms
               ───→────────────────→
                       결과 반환
```

**시각 효과:**
- 라인은 **빛나는 파이버 옵틱** 스타일
- 메시지가 이동할 때 **파티클이 라인을 따라 흐름**
- 색상: 요청=파란, 응답=초록, 에러=빨간
- 라인 두께 = 데이터 크기
