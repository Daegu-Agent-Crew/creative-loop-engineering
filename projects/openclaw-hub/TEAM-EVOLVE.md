# Team Evolve - OpenClaw Hub v1 → v2

기존 index.html을 읽고 각 모듈별로 개선하라.

## 기존 상태
- 파일: index.html (182KB, 6,548줄)
- 체크리스트: 49/52 통과
- 미달성 3항목: 3탭 명확한 표시, 워크플로우 갤러리 카드 뷰, 스크롤투탑

## 모듈 분할

### Teammate A: Office 영역 개선
- ❌ 3탭 복구: 네비게이션에 'Office' / 'Workflows' / 'Communicate' 명확히 표시
  - 각 탭은 버튼 형태, 클릭 시 해당 섹션만 display:block, 나머지 display:none
  - 활성 탭은 강조 스타일 (밑줄 또는 배경색)
  - 탭 전환 JS 로직
- 아이소메트릭 오피스 뷰 심화:
  - 에이전트 아바타에 상태별 CSS 애니메이션 (idle: 호버, working: 바운스, error: 흔들림)
  - 방 전환 시 페이드 트랜지션
  - 오피스 배경에 앰비언트 파티클 (먼지/빛 입자)
- 데스크 상호작용 개선:
  - 데스크 호버 시 글로우 효과
  - 데스크 위 모니터에 "화면" 표시 (CSS로 작은 창)
- 출력: evolve-a.html

### Teammate B: Workflows 영역 개선
- ❌ 워크플로우 갤러리 카드 뷰 복구:
  - 15개 워크플로우를 카드 그리드로 표시 (3열)
  - 각 카드: 이모지 아이콘, 이름, 카테고리 뱃지(색상 코딩), 스케줄, 활성 토글
  - 카드 호버: scale(1.02) + shadow
  - 카드 클릭 → 상세 모달
- 스케줄 타임라인 개선:
  - 현재 시간 빨간 세로선 (실시간 업데이트)
  - 워크플로우 마커에 이모지 + 이름 툴팁
  - 클릭 → 해당 워크플로우 하이라이트
- 카테고리 필터 개선:
  - 버튼 스타일 필터 (필요: 생산성/건강/스마트홈/재정/라이프스타일/전체)
  - 활성 필터 강조 + 카운트 표시
- 커스텀 생성 개선:
  - 폼 유효성 검사
  - 시간 선택기를 크론 대신 직관적인 시/분 선택으로
- ❌ 스크롤투탑 버튼 추가:
  - 우측 하단 원형 버튼
  - 300px 이하 스크롤 시 등장 (fade-in)
  - 클릭 → 부드러운 최상단 이동
- 출력: evolve-b.html

### Teammate C: Communication + Dashboard 개선
- 채팅 심화:
  - 에이전트가 "생각 중..." 타이핑 인디케이터
  - 메시지에 읽음 표시 (체크마크)
  - 이미지 전송 시뮬레이션 (placeholder)
- 툴콜 로그 심화:
  - 툴별 아이콘 (search=🔍, read=📄, exec=⚡, write=✏️)
  - 성공=초록, 실패=빨간, 대기=노랑 상태 색상
  - 레이턴시 바 (짧은=초록, 긴=빨간)
- 대시보드 개선:
  - Canvas 라인 차트: 시간대별 에이전트 활동량
  - Canvas 도넛 차트: 워크플로우 카테고리별 비율
  - 실시간 업데이트 애니메이션 (5초마다 데이터 변화)
- 알림 시스템:
  - 토스트 알림 (우측 상단 슬라이드인)
  - 알림 타입: 성공(초록), 정보(파랑), 경고(노랑), 에러(빨간)
  - 자동 5초 후 사라짐
- 출력: evolve-c.html
