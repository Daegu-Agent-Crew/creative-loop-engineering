## 🎯 개선 목표

### 1. 던전 맵 시각적 완성도 높이기
- 각 층 사이에 "계단" 또는 "통로" 시각 표현 (화살표 또는 연결선)
- 층마다 다른 배경 그라데이션 색상
  - ① INGESTION: 초록빛 (rgba(76,175,80,.05))
  - ② CONTROL: 주황빛 (rgba(255,152,0,.05))
  - ③ AGENT: 보라빛 (rgba(156,39,176,.05))
  - ④ EXECUTE: 시안빛 (rgba(0,188,212,.05))

### 2. 노드 디자인 개선
- 각 노드에 아이콘 이모지 크게 표시 (28px)
- 노드 안에 이름도 표시
- 노드 크기: 100x60px, 둥근 모서리 12px
- 비활성: 투명 배경 + 점선 테두리
- 활성: 불투명 배경 + 실선 + glow

### 3. 캐릭터 개선
- 💡 이모지 말고 CSS 원형 + glow로 구현
- 이동 경로를 dashed 선으로 표시 (남은 경로)
- 지나온 경로는 실선 + 밝은 색

### 4. 시뮬레이션 디테일
- 각 단계마다 상태창이 실시간 업데이트
  - HP: 단계 진행률 (총단계/현재단계 * 100)
  - MP: 남은 토큰 (랜덤 감소)
  - XP: 누적
  - 툴호출: 카운트업
- 보스 시나리오에서 대기 시간에 "로딩 바" 애니메이션

### 5. 미니맵 개선
- 실제 맵 축소판 (단순한 박스 4개)
- 현재 층에 깜빡이는 점

### 6. 라이브 모드 개선
- 연결 상태를 초록/빨간 점으로 표시
- WebSocket 연결 시 실제 이벤트를 받아 노드 활성화
- OpenAI API로 메시지 전송 후 응답을 시뮬레이션

### 7. 모바일 최적화
- 480px 이하: 사이드바를 하단 탭으로 변경
- 노드 크기 축소
- 폰트 크기 축소

### 8. 추가 애니메이션
- 층 진입 시 화면이 해당 층 색상으로 살짝 번쩍
- 완료 시 화면 중앙에 "퀘스트 완료!" 배너 (fadeIn + fadeOut)
- 연결선 위를 움직이는 작은 점 (CSS animation, SVG 불필요)
