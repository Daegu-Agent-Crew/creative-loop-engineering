# OpenClaw Visualizer v4 - "AI의 내부를 들여다보세요"

v3(index.html)를 기반으로 사람들이 AI 에이전트에 궁금해하는 것을 시각적으로 답하는 웹 앱.

---

## 📋 기반 파일

`$(pwd)/index.html` - **반드시 이 파일을 읽고 수정하세요. 삭제 금지.**

---

## 🎯 핵심 컨셉: "AI의 내부를 들여다보세요"

사용자가 AI 에이전트에 대해 가장 궁금해하는 3가지를 시각화:
1. **"내 메시지는 어떻게 처리되나요?"** → 메시지 여행 모드
2. **"왜 그 툴을 선택했나요?"** → 의사결정 트리
3. **"얼마나 비용이 들었나요?"** → AI 지갑

---

## 📋 구현할 것

### 1. 🎬 "내 메시지 여행하기" 모드 (핵심 신규)

기존 시뮬레이션을 대폭 개선. 사용자가 직접 메시지를 입력하면, 그 메시지가 AI를 거쳐 답변까지 도달하는 과정을 애니메이션으로 보여줌.

#### 화면 구성:
```
┌─────────────────────────────────────┐
│ 💬 메시지 입력: [_____________] [▶] │
├─────────────────────────────────────┤
│                                     │
│   ① 텔레그램 📨                     │
│   "내일 서울 날씨 알려줘"           │
│        │                            │
│        ▼                            │
│   ② 라우터 🔄                       │
│   "메인 세션으로 라우팅"             │
│        │                            │
│        ▼                            │
│   ③ AI 뇌 🧠                        │
│   "추론 중..."                       │
│   생각: 날씨 질문 → 검색 필요        │
│        │                            │
│        ▼                            │
│   ④ 검색 도구 🔍                    │
│   web_search("서울 내일 날씨")       │
│   결과: 맑음, 15°C                  │
│        │                            │
│        ▼                            │
│   ③ AI 뇌 🧠                        │
│   "답변 생성 중..."                  │
│        │                            │
│        ▼                            │
│   ① 텔레그램 ✅                     │
│   "내일 서울은 맑음, 15°C입니다"    │
│                                     │
├─────────────────────────────────────┤
│ 💰 토큰: 342개 | ⏱️ 2.3초 | 🛠️ 2회  │
└─────────────────────────────────────┘
```

#### 미리 만들 시나리오 (4개):
1. **"날씨 알려줘"** (초보) - 6단계, 1번 툴호출
2. **"이미지 분석해줘"** (중급) - 8단계, 2번 툴호출, vision
3. **"게임 만들어줘"** (보스) - 12단계, 5번 툴호출, 랄프톤
4. **"리마인더 설정해줘"** (시간) - 10단계, 3번 툴호출, cron

각 시나리오에서 **각 단계가 왜 필요한지** 한 줄 설명 표시.

### 2. 🧩 "AI의 생각 보기" (의사결정 트리)

노드를 클릭하면 그 노드에서 **왜 그 결정을 내렸는지** 툴팁/패널로 보여줌.

```
🔍 검색 노드 클릭 시:
┌────────────────────────┐
│ 🤔 왜 검색했을까요?     │
│                        │
│ AI가 분석한 내용:       │
│ "날씨" + "서울" →      │
│ 실시간 정보 필요함      │
│ → 웹 검색 선택         │
│                        │
│ 다른 선택지:            │
│ ❌ 메모리 검색 (과거X)  │
│ ❌ 파일 읽기 (관련없음)  │
│ ❌ 코드 실행 (불필요)   │
└────────────────────────┘
```

### 3. 💰 "AI 지갑" (실시간 통계 바)

화면 하단에 고정된 통계 바:
```
┌──────────────────────────────────────────────────┐
│ 🪙 토큰: 342   💵 $0.002   ⏱️ 2.3초   🛠️ 2회   │
│ ████████░░░░░░░░░░░░  context 사용률 23%        │
└──────────────────────────────────────────────────┘
```

시나리오 진행 중 실시간으로 증가하는 카운터 애니메이션.

### 4. 📖 "AI 설명서" 탭 (신규)

상단에 3개 탭:
- **🎬 메시지 여행** (기본)
- **📖 AI 설명서**
- **⚙️ 설정**

"AI 설명서" 탭 내용:
```
Q: AI는 어떻게 "생각"하나요?
A: 4단계로 생각합니다:
   1. 수신: 사용자 메시지를 받음
   2. 이해: 무엇을 원하는지 파악
   3. 도구: 필요한 도구를 선택하고 실행
   4. 응답: 결과를 바탕으로 답변 생성

Q: 왜 같은 질문에 다른 답이 나오나요?
A: AI는 확률적으로 다음 단어를 예측합니다.
   온도(temperature) 설정에 따라
   창의성과 일관성의 균형이 달라집니다.

Q: AI의 기억은 어떻게 작동하나요?
A: 두 가지 기억이 있습니다:
   - 단기: 대화 내용 (context window)
   - 장기: 파일에 저장 (MEMORY.md)

Q: 툴이 뭔가요?
A: AI가 직접 행동할 수 있는 도구입니다:
   🔍 검색, 📁 파일 읽기, ⚙️ 코드 실행,
   💾 메모리 저장, 📨 메시지 전송 등
```

FAQ 8~10개, 아코디언 스타일로 펼치기/접기.

### 5. ⚙️ 설정 탭
- 애니메이션 속도 (0.5x/1x/2x/3x)
- 소리 on/off
- 테마 (다크/라이트)
- 언어 (한국어/English) - 영어는 UI 텍스트만 번역

### 6. 모바일 최적화
- 480px 이하: 세로 스크롤로 단계별 표시 (가로 흐름 → 세로 흐름)
- 바텀 시트로 "AI의 생각" 표시
- 통계 바 축소

### 7. 입장 애니메이션 (신규)
첫 방문 시:
```
"AI의 내부를 들여다보세요" (2초)
→ RPG 맵이 점진적으로 나타남 (1초)
→ 캐릭터가 등장하고 손을 흔듦 (1초)
```
